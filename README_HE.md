# IMPACT SO

פרויקט נפרד למיתוג ולאתר IMPACT SO.

## מבנה

- `CLAUDE.md` — הוראות חובה לקלוד קוד עבור IMPACT SO בלבד.
- `site/impact-so-website.html` — קובץ האתר הנוכחי.
- `brand/` — רכיבי React, טוקנים, Loader ואייקונים.
- `public/brand/` — SVG שקופים מקוריים ללוגו ולטביעת האצבע.

## כלל עבודה

כל שינוי באתר או ברכיבי המותג מתחיל מקריאת `CLAUDE.md` ואז `brand/README_HE.md`.

הפרויקט הזה מופרד מהופה. לא משתמשים בנכסי הופה, בצבעי הופה או בהוראות העיצוב של הופה.
